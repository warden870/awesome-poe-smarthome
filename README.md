# awesome-poe-smarthome
A list of smarthome devices designed to be PoE-first.

## Cameras
Cameras on PoE is incredibly normal, so this list should only be cameras that include extra sensors.

| Camera | Extra Sensors |
|---|---|
| Unifi G4/G5/AI | Glass break and smoke detector noise sensors |

## Air Quality

| Device | Sensors |
|---|---|
| Awair Omni | Temp, Humidity, PM2.5, tVOC, Light, Noise |
| Room Alert 3E | Temp, Humidity |

## Presence
